# Product Context - Pernect

## Overview
Pernect（パーネクト）は、AI分析による自己発見プラットフォームです。MBTI診断をはじめとする多様な心理テストと自己分析ツールを提供するモバイルアプリケーションです。

## Vision
ユーザーが自分自身を深く理解し、より良い人生の選択をサポートすること。

## Target Users
- 自己理解を深めたい20-40代
- キャリアや人間関係に悩む社会人
- 自分の性格や強みを知りたい学生
- マッチングやチームビルディングに関心のある人々

## Core Value Propositions
1. **AIによるパーソナライズ分析**: 単なるテストではなく、AIが深い洞察を提供
2. **継続的な自己発見**: 一度きりではなく、継続的な分析と成長トラッキング
3. **直感的なUI/UX**: 美しく使いやすいモダンなデザイン
4. **プライバシー重視**: ユーザーデータの安全な管理

## Business Model
- **Free tier**: 基本的な診断テスト
- **Plus tier**: 追加の診断と詳細レポート
- **Pro tier**: AI分析、相性診断、キャリアアドバイス
- **Max tier**: 全機能アクセス、パーソナルコーチング

## Success Metrics
- DAU/MAU（日次/月次アクティブユーザー）
- 診断完了率
- サブスクリプション転換率
- ユーザーリテンション率
- NPS（ネットプロモータースコア）

## Current Status (2025-11)
- 認証システム（Clerk）実装済み
- 基本的なプロフィール機能実装済み
- ホーム/プロフィール/設定画面の基本UI完成
- 診断機能は未実装（モックデータのみ）
