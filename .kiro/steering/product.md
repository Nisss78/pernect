# Product Context - Pernect

## Overview

Pernect（パーネクト）は、AI分析による自己発見プラットフォームです。MBTI診断、エニアグラム、キャリア適性診断など多様な心理テストを提供し、ユーザーの自己理解を深めるモバイルアプリケーションです。

## Vision

ユーザーが自分自身を深く理解し、より良い人生の選択をサポートすること。

## Target Users

- **自己理解を深めたい20-40代**: 自分の性格や強みを知りたい人々
- **キャリアに悩む社会人**: 転職や キャリアチェンジを検討している人
- **人間関係を改善したい人**: コミュニケーションパターンを理解したい人
- **マッチング・チームビルディング関心層**: 相性診断や協働効率化を求める人

## Core Features

### 実装済み機能

- **心理診断テスト**:
  - MBTI診断（16タイプ全対応、詳細分析レポート）
  - エニアグラム診断（9タイプ）
  - キャリア適性診断（4タイプ: リーダー/サポーター/クリエイター/アナリスト）
- **パーソナライズ分析**: 強み・弱み・推奨事項を含む詳細な結果レポート
- **プロフィール管理**: ユーザー情報、MBTI結果、診断履歴の管理
- **プッシュ通知**: リマインダーや新テスト通知
- **多言語対応**: 日本語・英語

### 開発中・計画機能

- **プレミアムサブスクリプション**: Plus/Pro/Max階層
- **AIチャット**: パーソナルコーチング対話（プレースホルダー）
- **友達マッチング**: 相性診断・マッチング機能（プレースホルダー）
- **追加診断テスト**: 愛の言語、学習スタイル等


## Key Value Propositions

1. **科学的根拠**: 心理学に基づいた信頼性の高い診断
2. **AIによる深い洞察**: 単なる結果表示ではなく、具体的な行動提案
3. **継続的な自己発見**: 一度きりではなく、成長に合わせた再診断
4. **美しいUX**: 直感的で楽しい診断体験
5. **プライバシー重視**: セキュアなデータ管理

## Success Metrics

- **DAU/MAU**: 日次・月次アクティブユーザー数
- **診断完了率**: 開始した診断を最後まで完了する割合
- **サブスクリプション転換率**: 無料→有料ユーザー転換率
- **リテンション率**: 30日/90日ユーザー継続率
- **NPS**: ネットプロモータースコア（推奨度）

## Design Principles

1. **シンプルさ**: 複雑な心理学を誰でも理解できる形で提示
2. **ポジティブ**: 弱点指摘より成長機会の提示を重視
3. **継続性**: 一度の診断で終わらない長期的関係構築
4. **パーソナライズ**: 個人に合わせた体験とアドバイス
