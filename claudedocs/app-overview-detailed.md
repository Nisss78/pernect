# Pernect（パーネクト）完全アプリ概要書

## 1. アプリコンセプト

### 1.1 ビジョン
**「AI分析で自分自身を発見する」**

ユーザーが様々な心理テスト・診断を通じて自己理解を深め、その結果をAIとの対話や他者との相性診断に活用できるモバイルプラットフォーム。

### 1.2 アプリ名の由来
**Pernect** = **Per**sonality + Co**nnect**
- 性格（Personality）を理解し、自分や他者と繋がる（Connect）

### 1.3 ターゲットユーザー
- **主要ターゲット**: 20〜35歳の自己成長に関心のある層
- **サブターゲット**:
  - キャリアに悩む社会人
  - 人間関係を改善したい人
  - 自分の強み・弱みを知りたい人

### 1.4 コアバリュー
1. **自己発見**: 診断テストで自分を客観的に理解
2. **データ蓄積**: 診断結果を変数として保存・活用
3. **AI活用**: パーソナライズされたAI対話
4. **繋がり**: 相性診断で他者との関係性を可視化

## 6. スコアリングロジック

### 6.1 Dimension型（MBTI）
```typescript
// 8次元を追跡
const scores = { E: 0, I: 0, S: 0, N: 0, T: 0, F: 0, J: 0, P: 0 }

// 各回答でスコア加算
answers.forEach(answer => {
  const option = question.options.find(o => o.value === answer.selectedValue)
  scores[option.scoreValue]++  // "E" or "I" を加算
})

// 各軸で多い方を選択
resultType = [
  scores.E >= scores.I ? "E" : "I",
  scores.S >= scores.N ? "S" : "N",
  scores.T >= scores.F ? "T" : "F",
  scores.J >= scores.P ? "J" : "P"
].join("")  // → "ENFP"
```

### 6.2 Single型（エニアグラム等）
```typescript
// タイプごとにスコア合算
const scores = {}
answers.forEach(answer => {
  const option = question.options.find(o => o.value === answer.selectedValue)
  scores[option.scoreKey] = (scores[option.scoreKey] || 0) + option.scoreValue
})

// 最高スコアのタイプを選択
resultType = Object.entries(scores)
  .reduce((max, [type, score]) => score > max.score ? {type, score} : max)
  .type  // → "type1"
```
